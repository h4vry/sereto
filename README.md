# Security Reporting Tool

> Created with support of [NN Management Services, s.r.o.](https://www.nn.cz/kariera/en/it-hub/)

## Documentation

Install dependencies:

```sh
pip install "sereto[docs]"
```

Start local client:

```sh
mkdocs serve
```

## Development

### Tox

Run tests and linters:

```sh
tox
```

Format code:

```sh
tox -e format
```
